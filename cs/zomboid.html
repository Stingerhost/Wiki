Project Zomboid server

Server hry Project Zomboid je multiplayerová platforma, kde hráči společně bojují o přežití v otevřeném zombie apokalyptickém světě. Na serveru můžeš spolupracovat s ostatními hráči, budovat základny, sbírat zásoby a bránit se proti hordám nemrtvých. Cílem je přežít co nejdéle a užít si realistickou a napínavou hru plnou taktických rozhodnutí.
